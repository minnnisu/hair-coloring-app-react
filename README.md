## 🚀 AI 기반의 머리 염색 서비스**
>  Stable diffusion의 inpainting 기능을 활용하여 염색된 머리 사진을 생성해주는 서비스입니다.

## ✨ 서비스 기능

### [옵션 기능]
- 옵션을 통해 성별, 앞머리 여부, 머리 색상을 선택할 수 있습니다.
